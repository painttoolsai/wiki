Core Technical Insights
=======================

Paint Tool SAI 2 is a significant evolution over the original version, offering professional-grade features while maintaining its lightweight performance.

* **Advanced Canvas Limits:** - **64-bit version:** Supports up to **100,000 x 100,000px**.
  - **32-bit version:** Supports up to **10,000 x 10,000px**.
* **License Portability:** Users with a **SAI Ver.1 license** can unlock save/load features in Ver.2 for free.
* **New Layer Types:** Native support for **Text, Shape, and Linework** layers with path editing.
* **Perspective Rulers:** Support for 1 to 3 vanishing points created as dedicated layer objects.

System Requirements
===================

Ensure your PC meets these minimum specifications for a 2000px canvas:

* **OS:** Windows XP/Vista/7/8/8.1/10/11.
* **CPU:** Core 2 Duo 1.6GHz or better.
* **RAM:** Minimum 1GB (4GB+ recommended for 64-bit).
* **Storage:** 20GB+ free space for scratch disks.
* **Input:** Pen Tablet with WinTab or TabletPC API support.

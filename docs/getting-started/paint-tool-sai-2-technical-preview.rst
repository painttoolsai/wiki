Installation Requirements
=========================

Before downloading, ensure your system meets the minimum requirements:

* **Operating System:** Windows XP, Vista, 7, 8, 8.1, 10, or 11
* **Processor:** Intel Core 2 Duo 1.6GHz or equivalent
* **Memory:** 1GB RAM minimum (4GB+ recommended for 64-bit)
* **Storage:** At least 100MB for installation, plus scratch disk space

.. tip::
   For optimal performance with large canvases, we recommend the 64-bit version with at least 8GB RAM.

What's New in Technical Preview
===============================

The latest Technical Preview includes:

* **Enhanced Perspective Rulers** - Support for 1-3 vanishing points
* **New Layer Types** - Text, Shape, and Linework layers with vector editing
* **Improved Brush Engine** - Better bristle simulation and blending
* **Scatter Brushes** - New particle-based brush effects
* **Clipping Groups** - More flexible layer organization

Installation Steps
==================

1. Download the appropriate version (32-bit or 64-bit) from the official website
2. Extract the ZIP archive to your preferred location
3. Run ``sai2.exe`` directly from the extracted folder
4. No additional installation required

.. warning::
   SAI 2 does not use a traditional installer. Simply extract and run the executable.

License Activation
==================

To unlock save/load functionality:

1. Launch SAI 2 and note your **System ID**
2. Visit the license download page on the official website
3. Enter your purchase credentials and System ID
4. Download and install your license certificate

For detailed instructions, see :doc:`license-activation-transfer`.

Troubleshooting
===============

**Application won't start:**
   Ensure you have the latest Visual C++ Redistributable installed.

**Tablet not detected:**
   Verify your tablet driver supports WinTab or TabletPC API.

**Brushes don't work correctly:**
   Check if brush textures need updating. See :doc:`../tech/bristle-bitmap-corrector`.
